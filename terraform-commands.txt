Terraform commands to use :

To initiate and set up the terraform environment
terraform init 

To compare with real infrastructure and reviews the changes to be executed
terrform plan

To apply the changes 
terraform apply

To destroy or delete the applied changes
terraform destroy

To list the resources that were created
terraform state list

To skip the confirmation  and approve directly
terraform apply -auto-approve
terraform destroy -auto-approve

To delete/apply only specific tagets---
terraform destroy -target="aws_instance.web"

To  target multi blocks
terraform destroy -target="aws_instance.web" -target="aws_instance.aws_s3.storage"

To format .tf file with correct indentations and spaces
terraform fmt
To format  within the subfolders
terraform fmt -recursive

To check diferences
terraform fmt -check -diff

To to check if format of the .tf file is valid or not
terraform validate
 
To restrore from the drift seen
command : terraform plan -refresh-only
